# WEB-APP-SHOES-V2

Aplikasi web untuk membantu UMKM jasa cuci sepatu.

## Tech Stack

- Next.js (App Router)
- Prisma ORM
- PostgreSQL (Neon)
- NextAuth
- Tailwind CSS + shadcn/ui

## Main Features

- Customer order flow
- Admin dashboard
- Inventory & orders management
- Revenue & performance analytics

## Notes

This project is actively developed.
Some hooks/components are intentionally left empty
as part of planned refactoring.
